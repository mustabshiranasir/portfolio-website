# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, Bootstrap, and JavaScript. Created as a lab assignment for the Web Technologies course at COMSATS University Islamabad, Attock Campus.

## Live Demo

Hosted via GitHub Pages: `https://mustabshiratech.github.io/portfolio-website`

## About

This portfolio website showcases my skills, projects, and contact information as a Computer Science student. It features smooth scroll animations using the WOW.js library and a clean responsive layout built on Bootstrap 5.

## Sections

- **Home** — Introduction and hero banner
- **About** — Background and skills summary
- **Projects** — Showcase of academic and personal projects
- **Gallery** — Image gallery with visual work samples
- **Contact** — Contact form and social links

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure |
| CSS3 | Custom styling |
| Bootstrap 5 | Responsive grid and components |
| JavaScript | Interactive elements |
| WOW.js | Scroll-triggered animations |
| Font Awesome | Icons |

## Features

- Fully responsive — works on mobile, tablet, and desktop
- Scroll animations powered by WOW.js
- Sticky navbar with smooth scrolling
- Image gallery section
- Clean single-page layout

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/MustabshiraTech/portfolio-website.git
   ```
2. Open `index.html` in any browser — no setup required.

Or visit the live GitHub Pages link above.
